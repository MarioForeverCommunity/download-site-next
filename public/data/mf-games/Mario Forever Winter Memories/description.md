一切源于四五年前的一则MF社区春晚广告……
没错，Mario Forever Double Dictionary Route，一个可能没有多少人知道，绝大多数人都已经忘掉了的作品名字
在今天以另一个作品名的形式重生了：

**Mario Forever Winter Memories**

Mario Forever Winter Memories共有7个独立的关卡，这其中包含了Mario Forever Double Dictionary Route开坑时的一部分关卡，后来的关卡逐渐脱离了原有作品的设计思路（可能很少有人记得当时我开坑时说明的创作思路了：两个或多个主题完全不相同的关卡共用一个关卡标题，一个关卡标题能够很好的概括这几个关卡的主题），成为了各有特点的独立关卡，正因如此，本作品的游玩是非线性的（如果你玩过春游/酱游等多人合作蔚蓝地图模组，你应该会很熟悉）。

由于关卡较少，本作关卡的难度无法兼顾所有技术等级的玩家，故推荐技术水平较高的玩家游玩，关卡难度在游戏内也有标识。

更多内容不在此赘述，关卡截图可以在游戏内关卡开始前预览。欢迎反馈任何的游玩bug。

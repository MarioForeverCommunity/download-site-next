**前言**
自从2012年夏，吧主syzxchulun发布了9-1以来，永吧逐渐地进入了创作、讨论自制的作品的时代，也涌现出了AFS，MRI，MRI2，LGR等一系列广受赞誉的作品。在此对所有为MF同人作的引擎发展，设计理念进步做出贡献的吧友致以崇高的敬意！
再说Super MarioForever 2014（以下简称SMF2014），这部作品是由我于2014年Rainbow Engine发布后就计划制作的一部作品，并于2014年7月20日发布了1.0版本。发布之初得到了很高的关注，评价也都比较好。然而随着关卡设计理念的进步，制作技术的发展，SMF2014作为一部早期的作品，用现在的眼光来看，有着很多很多的问题。当初的SMF2014背景摆放、大地图制作，部分设计都不尽人意，而且因为赶工，更是削掉了一个世界。为了弥补之前没能实现、没能完成的部分，同时对作品内容做进一步的修改与完善，故特意对原作进行了重制，并追加了部分关卡，补全了十个世界，作为豪华版发布。
作品虽然名为豪华版，但是并不追求在特效，素材上过分的夺目，而是对之前版本的完善与提升，因此作品风格仍然保持之前的总体风格。虽然不敢说这个作品有多好，但至少较原来的版本有一些提升，并不失经典，不忘初心。

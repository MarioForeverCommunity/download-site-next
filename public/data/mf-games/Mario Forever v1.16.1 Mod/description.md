**Mario Forever v1.16.1 Mod** Release
A better way to experience the oldest version of the original MF. It's not much different, but it removes the requirement of having to install the game with files tied to C:\Windows. All v1.16.1 features are kept as is. Also, Human Lab has been ported over from v2.16. ~~and a few obscurely hidden secrets have been added.~~
This mod makes the game more compatible with newer hardware due to being ported to CTF2.5 and using Direct3D 9 render mode.

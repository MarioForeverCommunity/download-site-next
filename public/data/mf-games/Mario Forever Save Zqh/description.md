二、作品介绍
（1）作品新机制
本作引入了一个新的机制：等级机制。当你的分数达到一定数值后，可以升级，级别越高，HP上限越大，这里的HP与MFDE的性质类似，默认HP为HP上限的一半。如果受伤或挂命，会扣除一定分数。
而在本作中，道具均是二阶判定，顶道具问号时，视等级情况出现：顶不破、顶出红蘑菇、顶出绿果、顶出甜菜或顶出火力花，高级别补给状态下按A可以降至低一个级别的状态（但不会对级别造成任何影响）。
看起来很复杂吧？但其实种种平衡因素都经过考虑，整个机制有其实质性作用。
这里提前说明一下：在本作中，刷级是个比较有必要的概念。如果前期不尝试升级，后期的难度可能会超乎想象。与此同时，一般来说越靠后的关卡，分数奖励越丰厚。
与此同时，本作删除了命数和关卡时间机制，可无限挂命，通关用时也不作限制。
（2）作品规模
本作品总共包含5个世界30个关卡，其中主线世界4个，附加世界1个。难度非常大，主线世界的总体难度相当于8世界中偏难（或“难”）作品的后四世界，附加世界难度与本作中部难度相当。部分关卡需要的时间可能较长，还请耐心游玩。
每个主线世界都包含：①数字关卡（必须挑战）、②分支关卡C/P（包含一个挑战关卡和一个解谜关卡，二者通过其一即可）、③最终关卡F（必须挑战）。
作品采用自由选择的地图形式，在地图界面可以选择已解锁的关卡进入，通关后会重新回到地图界面。若要进入已经通过的关卡，则分数和级别都将被锁定，直至该关通关并回到地图。
附加世界需要解锁后才能在大地图中进入，解锁处在主线关卡2-4。
（3）作品特点
①在本作品中，数字关大多比较常规，比较传统；
②C关属于另类挑战，挑战的方式不同于一般的关卡，而是有一些新式的东西作为障碍或任务，难度较大；
③P关属于解谜关，类似于MFRL8-8。在制作本作的解谜关时，我力图谜题不落俗套，每个P关的谜题都包含一定的隐含逻辑，需要玩家主动发掘，如果不加分析地随意尝试，是基本不可能通过的。因此如果要玩，请先忘掉你对原有解谜关的认知。当然，这些谜题和MFDE式谜题没有任何关联，不要求你上知天文下知地理或者数理基础扎实，你只需要有一个喜欢解谜的大脑！
④F关是每个世界的最终关，难度比该世界的数字关大，较为硬核；
⑤附加世界意在展示我个人的BGM选取观念。每个关卡选取的BGM，其来源均与关卡内容大相径庭。乍一看会比较违和，但当你视BGM为BGM，不论其来源，不论其出处，不论其历史，BGM在你耳中只是一段音响时，你也许会觉得，它们，并没有那么违和。
三、操作说明
注：以下键位均是默认键位，可以在ini文件中更改。但是游戏中的按键提示均提示默认按键。因此如果更改键位，建议记住原键位，以便理解游戏中的提示。
上：上（主要用于进入开口朝下的水管）
下：下（主要用于大个子状态的下蹲、进入开口朝上的水管及降低跳跃高度）
左：左（主要用于向左移动及进入开口朝右的水管）
右：右（主要用于向右移动及进入开口朝左的水管）
跳跃：Z
加速/发射子弹/发射甜菜：X
重试：Q（用于自爆重玩）
降级：A（用于将补给降至低一个级别的状态）
特殊按键：S（在某些关卡中有用，相应的关卡中会有提示）
如果你想更改键位，可以在游戏所在目录下的ini文件下找到\[control\]字样，在其下方的若干键位中找到该键的功能，在等号后输入你想改成的按键就可以了（大小写均可）
例如我想将跳跃键更改为D，那么我只需要在"jump="后面输入"D"，即成为"jump=D"即可，不想改的其它键位不必理会。
默认的ini中，每个键的功能对应的等号后面为空，等号后为空则采取默认按键。
特别说明：在剧情中，按Enter可触发剧情进度，此按键不可更改。

五、关于本作品
本作品的制作方式其实是较为放飞自我的。在本作的制作中，我也遇到了一些其它成员的不同意见，但我坚持采取了我自己的部分想法。作品中也可能会有受众较窄，甚至引发争议的部分，如命数的取消，下键小跳和顶头失速的保留，BGM的选取理念，以及部分（在对欣赏要求较严的人看来）欣赏性并不好的独特设计。也许你并不喜欢它们，但这些设计我不会改动。因为我深知，创作不仅是为了大家的乐趣，更是要为自己创造乐趣，激发自己的创作热情。倘若一针一线都只是为他人甚至少数人而缝，却不含自己的一丁点想法，那这样的创作是没有血肉、没有灵魂的。如果你不喜欢我的风格，你大可选择不玩本作品；如果你愿意尝试我的独特风格，那么，享受它吧！
最后感谢大家一直以来对我的支持！

Lost Map X is a light MF fangame created in Ultra Edition, which has 6 levels.

#### But wait, what is Ultra Edition?

Ultra Edition (not engine, that's the one specifically used for SMMWW) is an MF game engine developed by dasasdhba, based on the Clickteam Fusion 2.5+ platform. It starts from zero and is NOT based on the Rainbow Engine. Seriously, it is by far the most advanced MF game engine on the CTF platform ever, with the pixel-perfect physical process and excellent architecture, not inferior to engines on the Godot platform. You can create an MF level on UE without the buggy and sticky physical experience that other CTF engines have, and easily develop complex and powerful new objects using the provided APIs. Sadly, the document and comments do not have an English version, and the fangames created on UE are also only popular among the CN community.
If you want to know more details about the CN community engine, see here: [https://www.marioforever.net/thread-3416-1-1.html](https://www.marioforever.net/thread-3416-1-1.html)

#### About this MF fangame

This fangame contains 6 levels with different themes, with save points for every single level, and the difficulty curve ranges from easy to moderate. It also has some detail improvements, like changing the Wii default sound effects in the UE engine back to original MFs (like getting coins), and better invincible star effects, etc. It is inspired by The Lost Map in the original MF and other fangames about it. Let's suppose that Lost Map world 1~8 for can't sleep's The Lost Map, and world 9 for Lost Map II in MFRb (it is similar to syzx world 9), then this fangame would be world 10, rewrite 10 as X is just Lost Map X.

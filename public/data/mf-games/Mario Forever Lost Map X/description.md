Q&A
Q：为什么叫Lost Map X？
A：如果can't sleep的The Lost Map作为World 1~8，而MFRb版中的Lost Map II作为World 9，那么这部作品就是World10，将10写成X之后刚好就是Lost Map X。
Q：你说这作品适合新人入坑，但为什么还是这么难？而且让新人直接入坑原版MF或那些大作不好吗？
A：想要让新人真正体验到MF关卡的魅力，则需要游玩有趣的MF关卡，而这些MF关卡通常都具有一定难度。并且，比起相对乏味的原版MF以及宏大的大作品，短小的新人训练或许更适合作为入门的开端。

## 技术栈

- **Node.js**: 20.x
- **前端框架**: Vue 3
- **编程语言**: JavaScript
- **构建工具**: Vite
- **包管理器**: npm

## 数据源

位于 `public/lists/` 下
- `list-original-mf.yaml`: 原版 Mario Forever 版本列表，对应 `index`
- `list.yaml`: Mario Forever 作品列表，对应 `mf-games`
- `list-mw.yaml`: Super Mario Worker Project 作品列表，对应 `mw-levels`

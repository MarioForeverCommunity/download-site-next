This catalog lists all games outside of Mario Forever, Mario and Non Mario, that Softendo/Buziol Games either created, repackaged, was credited in or is associated with. This also includes Banesoft games, who simply took the project files of Buziol's games and repackaged them as different games, Skell Zoft games which credit Buziol in some way and games made by a fan or partner that he distributed under his branding (mostly Flash games).

Due to them being made on older engines and older operating systems, some may not run properly or at all on your computer. As such, those usually require either an older computer or a virtual machine running an older operating system to properly run them, alongside any necessary drivers.

Each game is distinguished by Type (Mario, Non-Mario, Banesoft etc.), Engine (in what engine(s) the game was made) and Genre (what genre of videogame it belongs to, such as Puzzle, Find the Differences etc.) Some games share similar playstyles, and some are unfinished. Some games can also have multiple builds in the same engine, for example Super Mario Dark Dungeon's Flash builds, marked as (swf 1), (swf 2) and (swf 3).

Within each game's listing, there exists downloads to specific builds of the game. Each build has its own color-code to distinguish the types available:
- Blue: An installer for the game
- Orange: An installer + optional toolbar bundle for the game
- Green: A standalone exe file for the game that can be played anywhere on your computer.
- Purple: A portable zip/7z file which can be easily extracted and played, stripped of unnecessary advertising files.
- Red: A SWF of the game, exclusive to Flash games.

To find out more about these games, head to https://file.marioforever.net/mario-forever/games/softendo and https://file.marioforever.net/mario-forever/games/banesoft.
